## 📦 Dependencies

- Godot 4.6.2-stable
- Godot Export Template 4.6.2-stable
- GDScript Toolkit
- Git
- Windows 10 and above (x64)

Make sure these dependencies are installed before continuing. Using a package manager is recommended: [winget](https://github.com/microsoft/winget-cli), [chocolatey](https://chocolatey.org/), or [scoop](https://scoop.sh/).

**Note**: This project targets Windows 10 and above (x64) PCs only, and the development environment is set up entirely around the Windows ecosystem. For Linux/Mac users, please use a Windows virtual machine.

This project uses [Godot Unit Test](https://github.com/bitwes/Gut) 9.6.0 as its testing framework. Tests can be found under `res://tests/`.


## 🧰 Export and Deploy

To export and play the game locally for testing and debug, run the following command from the root directory:

- Release export:
    ```shell
    ./build.ps1 -c -m release
    ```
- Debug export:
    ```shell
    ./build.ps1 -c -m debug
    ```

You can find export output under `build/` directory. Artifact archive (`.zip` files) can be found in `artifact/` directory.

To clean exported files, run the following command:

```shell
./build.ps1 -c

```

For further information, use:

```shell
./build.ps1 -h

```

Final releases are done through tags on the GitHub repository. For each tag created, GitHub Actions will produce a build artifact. Attach this into the release.
